如何在mac的docker上运行GUI程序

  1. 提供mac 上本身不带有X window system support, 需要安装
    XQuartz以提供支持。
    推荐安装方法 brew cask install xquartz

  2. 配置Xquartz security: enable the connection from network port

  3. 配置Xhost（配置X host端）
    IP=$(ifconfig en0 | grep inet | awk ‘$1==”inet” {print $2}’)
    shell>> xhost + $IP
  4. 以-e 参数启动一个容器
    docker run -t -i -e DISPLAY=$IP:0 hasedon/centos6.5 /bin/bash
  5. 在该容器中便可以运行gui程序了。
